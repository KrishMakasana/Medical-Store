# 🏥 Medical Store Website

An interactive **Medical Store Website** designed to showcase digital marketing strategies and online healthcare services. This project demonstrates how a medical store can promote products and services using modern web technologies.

---

## 🚀 Features

* 🛒 Online Medical Store Interface
* 📝 Blog Section for Health Awareness
* 🎥 Video Marketing (Demo, Brand, Explanation Videos)
* 📧 Email Marketing Integration
* 📱 SMS & Mobile Marketing Concepts
* 🔍 Search Engine Based Marketing
* 🌐 Social Media Integration (Instagram, Facebook, WhatsApp, X, Google)
* 💻 Responsive and User-Friendly Design

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
